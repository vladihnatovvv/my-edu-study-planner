# MY-EDU Study Planner

Локальний планнер для відстеження прогресу по `Frontend`, `Backend` і
`Full Stack` темах.

## Як відкрити

1. Перейди в папку проєкту:

   ```bash
   cd /Users/ihnatovvladgmail.com/Documents/MY-EDU
   ```

2. Запусти локальний сервер:

   ```bash
   python3 -m http.server 4173
   ```

3. Відкрий у браузері:

   [http://127.0.0.1:4173](http://127.0.0.1:4173)

## Що вміє

- покрокові секції по навчальному маршруту
- чекбокси для відмітки вивченого
- загальний прогрес і прогрес по секціях
- фільтр `Усе / Ще вивчити / Вивчено`
- збереження стану в браузері через `localStorage`
